# CHANGELOG

## Unreleased

+ [调整] 移除首页右上角版本号和 GitHub 快捷入口。
+ [调整] 管理后台系统设置新增独立“注册邮箱”Tab，集中配置注册开关、邮箱验证码和 SMTP。
+ [调整] 管理后台易支付充值配置支持勾选支付宝、微信支付渠道，用户端只展示已启用渠道。
+ [新增] 管理后台模型渠道支持配置图片 1K/2K/4K 分辨率路由，图片生成和图片编辑会按请求档位选择渠道。
+ [修复] 优化画布资源引用输入框光标显示、连续 `@` 触发和 `@图1`、`@视1`、`@音1` 候选匹配。
+ [修复] 图片生成失败节点保留 prompt 时支持重新打开编辑面板修改后再生成。
+ [修复] 图片生成失败后保留输入框中的原始 prompt，已生成图片再次打开编辑面板时会自动回填历史 prompt，并修正选择多张生成时多创建 1 个图片节点的问题。
+ [修复] 画布从连接菜单选择“文本生成”后会打开文本生成输入面板，空文本节点不再显示“生图”快捷入口。
+ [优化] 画布视频生成支持 Veo Omni Flash/Video Edit 的参考视频和参考图 JSON 请求，并将画布默认生图张数调整为 1 张。
+ [优化] 云端渠道生图/图生图改为异步任务轮询，降低 Cloudflare Tunnel 长请求中断影响。
+ [修复] 修复 Next.js standalone 启动后静态资源缺失导致公网页面无样式的问题。
